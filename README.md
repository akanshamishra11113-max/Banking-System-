# 🏦 Banking System (SQL Project)

##  Overview

This project is a simple **Banking System Database** built using SQL.
It demonstrates core database concepts like tables, relationships, triggers, and stored procedures.

---

##  Technologies Used

* SQL (MySQL)
* Relational Database Concepts

---

## Database Structure

### Tables:

* **Customers** → Stores customer details
* **Accounts** → Stores account balance linked to customers
* **Transactions** → Records deposits and withdrawals

---

##  Features

✔️ Create and manage customers
✔️ Maintain account balances
✔️ Record transactions (Deposit / Withdraw)
✔️ Automatic balance update using **Trigger**
✔️ Money transfer using **Stored Procedure**
✔️ Basic queries for analysis

---

## Trigger Used

* Automatically updates account balance after each transaction

---

## Stored Procedure

* `TransferMoney` → Transfers money between two accounts

---

## How to Run

1. Open MySQL / SQL environment
2. Copy and paste the SQL file
3. Run the script
4. Execute test queries

---

##  Sample Queries

* View account balances
* View transaction history
* Find top customer by balance
